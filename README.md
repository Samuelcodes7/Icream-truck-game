#  ICream Truck Game 2D

A simple 2D Unity game created to revisit Unity fundamentals and improve my C# programming skills.

---

## About

This project is a recreation of one of the very first games I built when I started learning game development. I recreated it from scratch to refresh my knowledge of Unity while practicing cleaner code and better project organization.

Although the game is simple, it helped reinforce the core concepts of Unity and C# that are essential for building larger projects.

---

## Features

- Player movement
- Rigidbody2D physics
- Collision detection
- Trigger zones
- Boost pickups
- Simple game logic

---

## Built With

- Unity 6
- C#
- Unity 2D Physics
- Visual Studio

---

## What I Learned

Throughout this project, I revisited, practiced and improved my understanding of:

- C# scripts
- Player movement using Rigidbody2D
- Collision and trigger detection
- Using Unity Tags
- Organizing Unity projects
- Building modular gameplay systems
